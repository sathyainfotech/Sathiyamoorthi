Vision Projects - Object Tracking

Title

Development of Visual Object Tracking and Detection using Machine Learning and Deep Learning Technique

Overview

Deep correlation filter-based object tracking with ResNet feature extraction combines the efficiency of correlation filters and the powerful representation capabilities of deep learning. In this approach, deep features are extracted from the layers of ResNet, which capture high-level semantic information about the target object. These features are fed into the correlation filter, enhancing its ability to distinguish the object from complex backgrounds and handle variations in appearance. ResNet’s deep features help improve robustness against occlusion, scale changes, and background clutter, while the correlation filter offers efficient real-time tracking. This hybrid strategy leverages the discriminative power of ResNet and the speed of correlation filters, making it highly effective for tracking in challenging visual conditions.

Key Rolls

  • Experienced in machine learning and deep learning techniques

  • Proficient in deep feature extraction (VGGNet, ResNet Models, ext.)

  • Extensive knowledge acquired from Python, Tensorflow, PyTorch.

  • Proficient in software tools (MATLAB, Jupyter Notebook, Google Colab, CUDA GPU Activation).

Summary

In conclusion, deep correlation filter-based tracking using ResNet features offers a robust and efficient solution for object tracking in complex scenarios. By combining ResNet's ability to extract high-level, discriminative features with the speed and simplicity of correlation filters, this approach addresses challenges such as occlusion, scale changes, and background clutter. The synergy between these methods ensures accurate and real-time tracking, making it well-suited for dynamic and demanding computer vision applications.

Vision Projects - 3D Pose Estimation

Title

Development of 3D Human Pose Estimation using Transformer and Graph Neural Network Techniques

Overview

3D human pose estimation aims to predict the three-dimensional coordinates of human body joints from 2D images or video inputs. It has critical applications in fields such as motion capture, augmented reality, human-computer interaction, and sports analytics. Recent advancements leverage deep learning methods, including Transformers, which capture global dependencies within visual data, and Graph Neural Networks (GNNs), which effectively model the spatial and structural relationships between body joints. These techniques address challenges like occlusions, complex poses, and variations in viewpoint, enabling robust and accurate pose predictions. Ongoing research enhances pose estimation accuracy, generalizability, and efficiency, making systems more reliable in real-world scenarios.

Key Rolls

  • Specialized in Transformers and Graph Neural Networks Techniques.

  • Experienced in TensorFlow and PyTorch for developing deep learning models.

  • Proficient in software tools (Python, Visual Studio, Notepad++, CUDA GPU Activation).

  • Proficient in 3D human pose estimation filed.
